# code for LPC810
The program work on the LPC810 (LPC810M021FN8).
The LPC810 is an ARM Cortex-M0+ based, low-cost 32-bit MCU from NXP.

## Usage

### How to create ELF file

    cd PATH_TO_TREE_TOP/
    ./configure.sh EXAMPLE_DIRECTORY_NAME
    cd out/
    make

### How to clean up

    cd PATH_TO_TREE_TOP/
    ./configure.sh clean
